## Deploying to GitHub Pages

1. Create a new GitHub repository (e.g. `bk-breakfast-menu`)
2. Upload all files from this folder to the repository
3. Go to **Settings → Pages**
4. Set **Source** to `main` branch, root `/`
5. Click **Save** – your site will be live at `https://yourusername.github.io/bk-breakfast-menu/`

## Custom Domain (burgerkingbreakfastmenu.co.uk)

1. In your GitHub repo, go to **Settings → Pages → Custom domain**
2. Enter `burgerkingbreakfastmenu.co.uk`
3. In your domain registrar's DNS settings, add:
   - `A` records pointing to GitHub Pages IPs
   - Or a `CNAME` record pointing to `yourusername.github.io`
